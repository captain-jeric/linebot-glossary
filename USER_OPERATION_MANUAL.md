# LINE 翻译机器人用户操作说明书

本说明书面向 LINE 使用者，说明如何在私聊、群聊和多人聊天室中使用翻译机器人。本文不包含后台管理、充值开通或系统部署操作。

## 1. 使用前准备

使用前，请先确认管理员已经为你的 LINE 账号开通权限。

如果还没有开通，可以在 LINE 私聊机器人发送：

```text
userid
```

机器人会回复你的 `USERID`。请把这串 `USERID` 发给管理员开通。

如果已经开通，发送 `userid` 时会看到：

```text
USERID：Uxxxxxxxxxxxxxxxx
发送 /usage 查看额度。
```

## 2. 私聊翻译

在 LINE 中打开与机器人的私聊窗口，直接发送要翻译的文字即可。

默认情况下，机器人会按照你的默认语言设置进行互译。例如默认设置为 `中文 ↔ 泰文` 时：

- 发送中文，机器人回复泰文。
- 发送泰文，机器人回复中文。
- 发送其他语言，机器人会翻译成默认语言。

示例：

```text
你好，请问这个多少钱？
```

机器人会回复对应的翻译结果。

## 3. 群聊翻译

把机器人加入 LINE 群聊或多人聊天室后，已开通权限的用户在群里发言，机器人会自动把当前群聊绑定到该用户账号。

绑定后：

- 群内成员发送普通文字时，机器人会自动翻译。
- 群内翻译会使用绑定账号的额度。
- 一个群聊或多人聊天室同一时间只会绑定一个付费账号。
- 群聊可以单独设置翻译语言，不影响你的私聊默认语言。

如果群聊没有反应，可能是当前群聊还没有绑定账号。请让已开通权限的用户在群内发送一条普通消息，或发送：

```text
/status
```

## 4. 查看自己的 USERID

发送：

```text
userid
```

或：

```text
/userid
```

用途：

- 新用户把 `USERID` 发给管理员开通权限。
- 已开通用户核对当前 LINE 账号。

## 5. 查看额度

发送：

```text
/usage
```

机器人会回复：

- 账号名称
- 当前状态
- 有效期
- 总购买字符
- 已用字符
- 剩余字符
- 当前扣费模式

说明：

- 双语模式：普通消息按输入字符数扣额度。
- 三语模式：普通消息按输入字符数 x 2 扣额度。

## 6. 查看当前翻译状态

发送：

```text
/status
```

机器人会回复当前翻译状态，包括：

- 当前是在私聊、群聊还是多人聊天室。
- 当前账号是否有效。
- 当前使用用户默认配置，还是当前群聊配置。
- 当前是双语模式还是三语模式。
- 当前默认语言和互译语言。
- 群聊自动翻译是否开启。

## 7. 切换双语翻译语言

发送：

```text
set 语言1 语言2
```

示例：

```text
set zh th
```

表示切换为 `中文 ↔ 泰文`。

```text
set th ja
```

表示切换为 `泰文 ↔ 日文`。

规则：

- 第一种语言是默认语言。
- 第二种语言是常用互译语言。
- 收到第一种语言时，会翻译成第二种语言。
- 收到第二种语言时，会翻译成第一种语言。
- 收到其他语言时，会翻译成第一种默认语言。

在私聊中发送 `set zh th`，会修改你的个人默认语言。

在群聊中发送 `set zh th`，只会修改当前群聊的语言设置，不影响你的私聊或其他群聊。

## 8. 切换三语模式

发送：

```text
set 3lang
```

三语模式支持：

```text
中文 / 泰文 / 缅文
```

开启后，发送中文、泰文或缅文时，机器人会翻译成另外两种语言。

注意：三语模式下，普通消息按 `输入字符数 x 2` 扣额度。

如果想切回双语模式，可以发送：

```text
set zh th
```

也可以换成其他双语组合，例如：

```text
set th en
```

## 9. 指定翻译成某种语言

如果不想使用当前默认语言，可以用指定翻译命令：

```text
/语言代码 内容
```

常用示例：

```text
/TH 你好
/ZH สวัสดี
/MM 你好
/EN 你好
/JP 你好
/KO 你好
```

支持的指定翻译命令：

```text
/TH 内容    翻译成泰文
/MM 内容    翻译成缅文
/ZH 内容    翻译成中文
/TW 内容    翻译成繁体中文
/EN 内容    翻译成英文
/JP 内容    翻译成日文
/DE 内容    翻译成德文
/FR 内容    翻译成法文
/ES 内容    翻译成西文
/RU 内容    翻译成俄文
/MS 内容    翻译成马来文
/KO 内容    翻译成韩文
/ID 内容    翻译成印尼文
/VI 内容    翻译成越南文
/HI 内容    翻译成印地文
/AR 内容    翻译成阿拉伯文
```

指定翻译只翻译成你指定的目标语言，不受当前双语或三语模式影响。

## 10. 开启或关闭群聊自动翻译

在群聊中，已开通权限且账号可用的用户可以发送：

```text
set on
```

开启当前群聊自动翻译。

发送：

```text
set off
```

关闭当前群聊自动翻译。

关闭后：

- 群内普通消息不会自动翻译。
- 仍然可以使用 `/TH 内容`、`/ZH 内容`、`/MM 内容` 等指定翻译命令。

## 11. 查看群聊 ID

在群聊或多人聊天室中发送：

```text
/groupid
```

机器人会回复当前群聊 ID 或聊天室 ID。

当需要管理员协助处理群聊绑定、扣费账号或翻译设置时，可以把这个 ID 发给管理员。

## 12. 不会触发翻译的情况

以下情况机器人通常不会翻译：

- 发送的是图片、语音、贴图、文件等非文字内容。
- 消息为空白。
- 消息以 `!` 开头。
- 消息以 `//` 开头。
- 群聊自动翻译已关闭，且没有使用 `/TH 内容` 这类指定翻译命令。
- 当前账号未开通、已暂停、已过期或额度不足。
- 机器人无法识别源语言。
- 源语言和目标语言相同。

## 13. 常见提示说明

### 当前账号尚未开通权限

表示你的 LINE 账号还没有开通。请发送：

```text
userid
```

然后把机器人回复的 `USERID` 发给管理员。

### 账号已暂停

表示当前账号暂时不能使用。请联系管理员处理。

### 账号有效期已过

表示账号已经过期。请联系管理员续费或充值。

### 当前字符余额不足

表示剩余额度不够本次翻译。请联系管理员充值流量。

### 暂时无法完成翻译

可能是翻译服务临时失败、语言识别失败，或源语言和目标语言相同。可以稍后重试，或使用 `/TH 内容` 这类指定翻译命令。

## 14. 快速命令表

```text
help        查看帮助
/help       查看帮助
userid      查看 USERID
/userid     查看 USERID
/groupid    查看当前群聊ID
/usage      查看额度
/status     查看当前状态
set on      开启群聊自动翻译
set off     关闭群聊自动翻译
set 3lang   开启中文 / 泰文 / 缅文三语模式
set zh th   切换为中文 ↔ 泰文
set th en   切换为泰文 ↔ 英文
/TH 内容    指定翻译成泰文
/ZH 内容    指定翻译成中文
/EN 内容    指定翻译成英文
```

## 15. 建议使用方式

私聊使用建议：

- 日常一对一翻译，直接发送文字即可。
- 临时指定目标语言时，使用 `/TH 内容`、`/EN 内容` 等命令。
- 不确定当前设置时，先发送 `/status` 查看。

群聊使用建议：

- 让负责扣费的已开通用户先在群里发言，完成群聊绑定。
- 群里多人日常交流时，保持 `set on`。
- 群里消息很多、不想全部翻译时，发送 `set off`，需要翻译时再用指定翻译命令。
- 群聊翻译语言和私聊语言可以分开设置。
